    fax_faxfile_send


    发送传真文件 

    	
  $retrun = fax_faxfile_send($account,$number,$filetype,$filebinary,$sendannounce)

    参数:
    $account 传真发送的帐户,如果不填写以匿名帐户方式发送.
    $number 对方传真的号码,拨号流程将通过拨出路由进行控制.
    $filetype 传真文件的类型,目前只支持填写tiff
    $filebinary 传真文件的类型的内容,binary格式
    $sendannounce 传真发送提示语音,建议默认3次

    返回数据: $retrun
    无

     

    fax_faxfile_download


    传真文件下载 

    	
  $retrun = fax_faxfile_download($faxid)

    参数:
    $faxid 传真队列编号

    返回数据: $retrun

  • binarystring 'filestream' : 二进制数据文件内容
  • string 'filename' : 文件名称
