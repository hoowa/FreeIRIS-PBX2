ROUTER呼叫规则函数包

 

作者 : Sun bing
版本 : 1.0
文件 : rpcpbx_router.php
说明 :
  ROUTER呼叫规则函数包
函数 :

  • router_list
  • router_get
  • router_add
  • router_edit
  • router_delete
  • router_recall

 

router_list


呼叫规则列表 

	
  $retrun = router_list($routerline)

参数:
$routerline 呼叫规则类型,1分机,2中继

返回数据: $retrun

  • array 'rules' : 呼叫规则列表结构

    •  

    router_get


    取得指定一条呼叫规则信息 

    	
  $retrun = router_get($routerid)

    参数:
    $routerid 呼叫规则编号

    返回数据: $retrun

  • array 'resdata' : 呼叫规则信息

    •  

    router_add


    增加一条呼叫规则 

    	
  $retrun = router_add($proirety,$createmode,$routerline,$routerdata)

    参数:
    $proirety 优先级别
    $createmode 0表示自动创建,1表示人工创建,2表示无法手动删除(优先级别是最低的)
    $routerline 0未分配,1内线路由,2外线路由
    $routerdata 路由其他数据

    返回数据: $retrun
    fri2标准返回

     

    router_edit


    编辑呼叫规则 

    	
  $retrun = router_edit($routerid,$routerdata,$routerline)

    参数:
    $routerid 呼叫规则编号
    $routerdata 路由被编辑数据结构
    $routerline 0未分配,1内线路由,2外线路由

    返回数据: $retrun
    fri2标准返回

     

    router_delete


    删除一条规则 

    	
  $retrun = router_delete($id)

    参数:
    $id 规则编号

    返回数据: $retrun
    fri2标准返回

     

    router_recall


    更新IVR菜单动作的优先级别 

    	
  $retrun = router_recall($routerid,$proirety)

    参数:
    $routerid 路由规则编号
    $proirety 新优先级别

    返回数据: $retrun
    fri2标准返回
